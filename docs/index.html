SmartRing

受罗技 Action Ring 启发的环形应用启动器 — 按下快捷键，圆环在光标周围展开，移动鼠标选择应用，松开即启动。

PyQt5 Windows Python 3.7+ MIT

功能

快捷键唤出

默认 F12，可配置任意组合键（Ctrl/Alt/Shift/Win），适配鼠标侧键映射。

光标环形菜单

应用图标圆形排列在光标周围，自动提取真实 .exe 图标，含柔和阴影。

鼠标指向选择

移动鼠标到对应扇区自动高亮，图标放大 + 标签高亮，松开即启动。

Hold / Toggle 双模式

Hold：按住唤出、松开启动。Toggle：按一下显示、再按隐藏。

开机自启

最高优先级设置项，随 Windows 自动启动，确保随时可用。

可自定义外观

8 种预设主题色 + 自定义色值，圆环尺寸、图标大小、背景色均可调，带实时预览。

键盘模拟器

设置面板内置模拟键盘，输入快捷键时自动高亮对应按键。

快捷键冲突提醒

检测 F12 等常用系统快捷键并提供冲突警告。

图形化设置界面

右键托盘图标打开设置面板，实时预览调整参数，保存后可预览真实圆环效果。

一键重置

托盘菜单提供「初始化」选项，两次确认后恢复出厂默认设置。

下载

SmartRing.exe

独立可执行文件，无需安装 Python

Latest Release GitHub

下载 SmartRing.exe 双击运行，首次启动自动弹出设置向导。

快速开始

方式 1：直接使用 EXE（推荐）

下载 dist/SmartRing.exe，双击运行。

方式 2：Python 源码

# 安装依赖
pip install -r requirements.txt

# 启动（无控制台窗口）
pythonw SmartRing.pyw

# 或双击 run.bat

配置项

配置项说明默认值
auto_start开机自启（最高优先级）false
hotkey快捷键f12
mode触发模式 (hold / toggle)hold
theme界面主题 (light / dark)dark
accent_color高亮主题色#0078D4
bg_color圆环背景色#1e1e26
ring_radius外环半径 (px)190
center_radius中心半径 (px)58
icon_size图标大小 (px)38
animation_duration动画时长 (ms)220
show_labels显示应用标签true

鼠标侧键设置

  1. 打开 Logitech G Hub / Options
  2. 选择你的鼠标
  3. 将侧键映射为键盘按键 F12（或你自定义的快捷键）
  4. 按下侧键即可唤出 SmartRing
